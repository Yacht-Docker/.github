## Fixing Common Yacht Setup Problems

- Yacht web UI not loading? Confirm the container is running, the mapped port is open, and the browser is pointed at the correct server address.  
- Yacht install fails? Check that Docker is installed correctly, your user has permission to manage Docker, and the required volume paths exist.  
- Yacht docker compose stacks not appearing as expected? Review compose formatting, environment variables, network names, and mounted configuration files.  
- Yacht templates do not deploy? Refresh template sources, verify image names, and confirm the Docker host can reach the needed registries.  
- Yacht update causes unexpected behavior? Read Yacht documentation, back up persistent data, and restart the service after pulling the newer image.  
- Yacht reverse proxy access breaks login or routing? Recheck proxy headers, HTTPS settings, container ports, and any path-based routing rules.
